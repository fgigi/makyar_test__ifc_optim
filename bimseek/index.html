BIMSeek: A content-based search engine for BIM models using IFC ontology

Meng Wanga, Yu-Shen Liua,b,c,*, Ge Gaoa,d, Ming Gua,b,c, Jun-Hai Yonga,b,c

aSchool of Software, Tsinghua University, Beijing, China
bKey Laboratory for Information System Security, Ministry of Education of China

Abstract:

With the popularity of Building Information Modeling (BIM) technology in Architecture, Engineering and Construction (AEC) industry field, BIM resources especially building product library models are growing rapidly on the World Wide Web; Nevertheless, existing web search engines place more attention on metadata-based BIM model document retrieval, neglecting the data stored in the content of BIM model itself, which contains more information than in the description document.

To address this issue, we presents a content-based search engine system utilizing IFC ontology, named BIMSeek, for retrieving web BIM models. BIMSeek transforms BIM models in different formats and from different sources to one standard ontology, and provides a unified retrieval way. Then the content of the BIM models can be conveniently retrieved. Our results show that BIMSeek will have great utility although there is potential for improvement.

Contributions:
-Compared to the traditional metadata retrieval of BIM models, we present a way to search into the content of BIM models, which contains more information.
-Compared to the single BIM resource web site, our system serves as a vertical search tool to search through different web sites in AEC industry.
-Unlike regular simple search of BIM models, our system provide a way to customize their requirement specifically. Both accurate numerical value search and text search is supported.
-In contrast with IFC and other format, our system stores data in standardized OWL, allowing the BIM data compatible with current W3C semantic web implementation and flowing outside the web site.
